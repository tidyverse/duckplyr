# Verbs not implemented in duckplyr

The following dplyr generics have no counterpart method in duckplyr. If
you want to help add a new verb, please refer to our contributing guide
<https://duckplyr.tidyverse.org/CONTRIBUTING.html#support-new-verbs>

## Unsupported verbs

For these verbs, duckplyr will fall back to dplyr.

- [`dplyr::add_count()`](https://dplyr.tidyverse.org/reference/count.html)

- [`dplyr::cross_join()`](https://dplyr.tidyverse.org/reference/cross_join.html)

- [`dplyr::do()`](https://dplyr.tidyverse.org/reference/do.html)

- [`dplyr::group_by()`](https://dplyr.tidyverse.org/reference/group_by.html)

- [`dplyr::group_indices()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`dplyr::group_keys()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`dplyr::group_map()`](https://dplyr.tidyverse.org/reference/group_map.html)

- [`dplyr::group_modify()`](https://dplyr.tidyverse.org/reference/group_map.html)

- [`dplyr::group_nest()`](https://dplyr.tidyverse.org/reference/group_nest.html)

- [`dplyr::group_size()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`dplyr::group_split()`](https://dplyr.tidyverse.org/reference/group_split.html)

- [`dplyr::group_trim()`](https://dplyr.tidyverse.org/reference/group_trim.html)

- [`dplyr::groups()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`dplyr::n_groups()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`dplyr::nest_by()`](https://dplyr.tidyverse.org/reference/nest_by.html)

- [`dplyr::nest_join()`](https://dplyr.tidyverse.org/reference/nest_join.html)

- [`dplyr::reframe()`](https://dplyr.tidyverse.org/reference/reframe.html)

- [`dplyr::rename_with()`](https://dplyr.tidyverse.org/reference/rename.html)

- [`dplyr::rows_append()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rows_delete()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rows_insert()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rows_patch()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rows_update()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rows_upsert()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`dplyr::rowwise()`](https://dplyr.tidyverse.org/reference/rowwise.html)

- [`generics::setequal()`](https://generics.r-lib.org/reference/setops.html)

- [`dplyr::slice_sample()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`dplyr::slice_tail()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`dplyr::slice()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`dplyr::ungroup()`](https://dplyr.tidyverse.org/reference/group_by.html)
