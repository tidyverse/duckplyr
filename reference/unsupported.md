# Verbs not implemented in duckplyr

The following dplyr generics have no counterpart method in duckplyr. If
you want to help add a new verb, please refer to our contributing guide
<https://duckplyr.tidyverse.org/CONTRIBUTING.html#support-new-verbs>

## Unsupported verbs

For these verbs, duckplyr will fall back to dplyr.

- [`add_count()`](https://dplyr.tidyverse.org/reference/count.html)

- [`cross_join()`](https://dplyr.tidyverse.org/reference/cross_join.html)

- [`do()`](https://dplyr.tidyverse.org/reference/do.html)

- [`group_by()`](https://dplyr.tidyverse.org/reference/group_by.html)

- [`group_indices()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`group_keys()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`group_map()`](https://dplyr.tidyverse.org/reference/group_map.html)

- [`group_modify()`](https://dplyr.tidyverse.org/reference/group_map.html)

- [`group_nest()`](https://dplyr.tidyverse.org/reference/group_nest.html)

- [`group_size()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`group_split()`](https://dplyr.tidyverse.org/reference/group_split.html)

- [`group_trim()`](https://dplyr.tidyverse.org/reference/group_trim.html)

- [`groups()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`n_groups()`](https://dplyr.tidyverse.org/reference/group_data.html)

- [`nest_by()`](https://dplyr.tidyverse.org/reference/nest_by.html)

- [`nest_join()`](https://dplyr.tidyverse.org/reference/nest_join.html)

- [`reframe()`](https://dplyr.tidyverse.org/reference/reframe.html)

- [`rename_with()`](https://dplyr.tidyverse.org/reference/rename.html)

- [`rows_append()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rows_delete()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rows_insert()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rows_patch()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rows_update()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rows_upsert()`](https://dplyr.tidyverse.org/reference/rows.html)

- [`rowwise()`](https://dplyr.tidyverse.org/reference/rowwise.html)

- [`setequal()`](https://generics.r-lib.org/reference/setops.html)

- [`slice_sample()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`slice_tail()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`slice()`](https://dplyr.tidyverse.org/reference/slice.html)

- [`ungroup()`](https://dplyr.tidyverse.org/reference/group_by.html)
